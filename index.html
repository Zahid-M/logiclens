How it works

Built on Mesocosm, SWECC's managed benchmark platform.

1
20 logic puzzles with deterministic correct answers — deductive ordering, syllogisms, and arithmetic constraint problems.
2
3 framings per puzzle — neutral (Person A/B/C), Western names, and diverse international names. Logically identical in every version.
3
Deployed on Mesocosm via mesocosm env submit and mesocosm run create, evaluating multiple frontier models.
4
Consistency scoring — a model scores 1 point only when it gives the same answer across all 3 framings, regardless of whether it's correct.
